# Linux Command Reference

Welcome to the Linux Command Reference! This repository provides a curated, readable, and easy-to-check list of popular and highly frequent Linux commands, organized by category. It also includes a quick-reference cheatsheet and some example shell scripts.

## Repository Structure

- **[`README.md`](README.md)**: This file, explaining the repository structure and acting as a table of contents.
- **[`cheatsheet.md`](cheatsheet.md)**: A quick, tabular reference to the most popular Linux commands for easy reading.
- **`doc/`**: Detailed documentation for commands, separated by categories.
  - [`file_management.md`](doc/file_management.md): File and directory operations (`ls`, `find`, `cp`, `mv`, `tar`, etc.).
  - [`text_processing.md`](doc/text_processing.md): Text and stream operations (`grep`, `awk`, `sed`, `cat`, etc.).
  - [`system_info.md`](doc/system_info.md): System monitoring and resource usage (`top`, `ps`, `df`, `du`, etc.).
  - [`networking.md`](doc/networking.md): Networking commands (`curl`, `wget`, `ping`, `ip`, etc.).
  - [`ros2.md`](doc/ros2.md): Robot Operating System 2 commands (`ros2 run`, `ros2 topic`, `colcon build`, etc.).
- **`scripts/`**: Example shell scripts demonstrating practical usage of various commands.
  - [`backup.sh`](scripts/backup.sh): An example script for backing up directories.
  - [`system_health.sh`](scripts/system_health.sh): An example script for checking system health (disk, memory).

## How to use

If you need a quick reminder of how a command works, check out the **[cheatsheet](cheatsheet.md)**.

If you are looking for more details, examples, and flags for specific commands, browse through the **`doc/`** directory.
